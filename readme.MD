
# VG Game

## VG Game is a 2D pokemon style game. In this game you can walk around city and battle with your pokemon.
Game link: https://vg-game.vercel.app/

### Technologys: 
HTML, CSS, JavaScript
### Libraries: 
GSAP to do animations and Howler to handle the sounds
